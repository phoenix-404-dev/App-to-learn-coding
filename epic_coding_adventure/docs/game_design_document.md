# Epic Coding Adventure - Game Design Document (GDD)

## 1. Game Overview
**Title:** Epic Coding Adventure
**Genre:** RPG / Educational
**Target Audience:** Aspiring developers, students, logic puzzle enthusiasts.
**Core Loop:** Explore Realm -> Encounter Challenge -> Write Code to Solve -> Earn XP/Loot -> Unlock Next Area.

## 2. Story & Setting
**The Kingdom of Code**: A digital dimension manifesting programming concepts as physical reality.
**The Protagonist**: "The Null Pointer" (Player) - a coder with potential but no memory.
**The Mentor**: Archmage of Syntax.
**The Antagonist**: "The Spaghetti Monster" and his "Bug Legion".

## 3. Level Design (The 6 Realms)

### Level 1: The Serpent's Garden (Python)
- **Theme**: Jungle, Ancient Ruins, Snakes.
- **Concepts**: Variables, Strings, Print, Basic Math.
- **Boss**: The Infinite Loop Python (Ouroboros).
- **Unlock**: "Snake Charmer's Flute" (Allows controlling flow).

### Level 2: The Web of Illusions (JavaScript)
- **Theme**: Neon City, Spiders, Asynchronous Glitches.
- **Concepts**: Functions, DOM Manipulation (simulated), Events, Async/Await.
- **Boss**: The Callback Hell Spider.
- **Unlock**: "Event Listener Lens" (See hidden triggers).

### Level 3: The Sharp Fortress (C#)
- **Theme**: Medieval Castle, Knights, Strict Structure.
- **Concepts**: Classes, Objects, Types, Methods.
- **Boss**: The Garbage Collector Golem.
- **Unlock**: "Shield of Encapsulation".

### Level 4: The Pointer Peaks (C++)
- **Theme**: Icy Mountains, Raw Power, Dangerous Terrain.
- **Concepts**: Pointers, Memory Management, References.
- **Boss**: The Segmentation Fault Dragon.
- **Unlock**: "Memory Wand" (Direct access).

### Level 5: The Virtual Island (Java)
- **Theme**: Tropical Archipelago, Factories, Coffee Plantations.
- **Concepts**: OOP, Inheritance, Interfaces, JVM Magic.
- **Boss**: The Abstract Factory Overlord.
- **Unlock**: "Blueprint Scroll" (Instantiate objects).

### Level 6: The Data Caverns (SQL)
- **Theme**: Underground Crystal Mines, Tables, Relational Ley Lines.
- **Concepts**: Select, Insert, Update, Joins.
- **Boss**: The Little Bobby Tables (Injection Phantom).
- **Unlock**: "Query Crystal" (Summon data).

## 4. Mechanics
- **Spellcasting**: Writing correct syntax casts the spell.
- **Mana**: Syntax Errors drain Mana (Health).
- **Inventory**: Code Snippets (Reusable functions).
- **XP**: Gained by efficiency and correctness.

## 5. UI/UX
- **HUD**: Health (Red), Mana (Blue), XP Bar (Green).
- **Code Editor**: Bottom half of screen, slides up during combat.
- **Dialogue**: Visual Novel style overlay.
